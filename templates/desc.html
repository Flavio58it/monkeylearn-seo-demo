How it works

Enter a URL (your page) and a keyword phrase (to query Google) and an engine ccTLD (e.g. .com, .co.uk, .ca, etc). MonkeyLearn grabs the results from page one of Google (top 10 results), extracts out the relevant keywords from each page, and then shows a comparison of those vs. the keywords on your page.

Under the hood

We use MonkeyLearn to empower this Keyword Comparison Extractor with machine learning:

  • We use the HTML to text extractor to extract the relevant text from the HTML of each page. This machine learning algorithm can be used to detect and remove the surplus "clutter" (boilerplate, templates) around the main textual content of a web page.
  • We use the Keyword Extractor to identify the relevant keywords from text in English. Keywords can be compounded by one or more words and are defined as the important topics in the content.
